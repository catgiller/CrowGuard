⚠️ PİTORESK AI - KARŞILAŞILAN SORUNLAR

Sürecin uzamasına ve tıkanıklıklara neden olan temel problemler:

1. GEMINI MODEL 404 HATALARI:
----------------------------
* En büyük engel: 'list_models.py' çıktısında görünmesine rağmen 'gemini-2.5-flash' ve 'gemini-1.5-flash' modelleri API tarafından "not found" (bulunamadı) hatasıyla reddedildi. Bu durum, kullanılan SDK versiyonu veya API anahtarının yetki bölgesiyle ilgili olabilir.

2. CORS POLİTİKASI ÇAKIŞMALARI:
-------------------------------
* Tarayıcı, localhost:3000 ve 127.0.0.1:3000 adreslerini farklı originler olarak gördüğü için API isteklerini blokladı. 'allow_origins' listesinde birden fazla değer olması bazı durumlarda tarayıcı tarafında "multiple values" hatasına yol açtı.

3. 503 SERVICE UNAVAILABLE:
---------------------------
* Yapay zeka servisinden yanıt alınamadığında (model hatası yüzünden), backend 503 hatası döndü. Bu durum bazen frontend tarafında CORS hatası gibi görünerek asıl hatayı maskeledi.

4. SDK VE API VERSİYON UYUMSUZLUĞU:
-----------------------------------
* 'google-genai' kütüphanesinin v1beta ve v1 versiyonları arasındaki model isimlendirme farkları (models/ ön eki gerekip gerekmemesi) stabiliteyi bozdu.
