- [ ] **"YZ" kısaltması:** Hero’da "YZ Destekli" — hedef kitle için "Yapay zeka destekli" daha anlaşılır olabilir.
- [ ] **İletişim formu:** Gönderim yalnızca `setTimeout` mock; gerçek API/e-posta entegrasyonu yok — kullanıcıya başarı mesajı yanıltıcı olabilir.
- [ ] **Şifremi unuttum:** `/login` içinde `href="#"` — sayfa veya akış yok.

### İşlev
- [ ] **Ürün analizi:** Tamamen mock veri; `lib/api.ts` ile backend analiz uçları bağlanmamış.
- [ ] **Akıllı asistan:** Kural tabanlı mock yanıtlar + `dangerouslySetInnerHTML`; `/advisor` API ve streaming yok.
- [ ] **Dashboard istatistikleri / geçmiş:** Sabit demo veriler; gerçek kullanıcı geçmişi API’den gelmiyor.
- [ ] **Hızlı arama (dashboard):** Input değeri analiz sayfasına aktarılmıyor; "Analiz Et" yalnızca route değiştiriyor.
- [ ] **Plan toggle (Ücretsiz/Pro):** Yalnızca UI demo; ödeme/abonelik backend’i yok.
- [ ] **Çıkış (sidebar footer):** Layout sidebar’ında çıkış yok; yalnızca üst barda — tutarlılık için sidebar’a `logout` eklenebilir.
- [ ] **Ölü kod:** `components/sidebar.tsx` (Framer Motion) hiç import edilmiyor; `dashboard/layout.tsx` kendi sidebar’ını kullanıyor — silinmeli veya birleştirilmeli.
- [ ] **`.env.example`:** `NEXT_PUBLIC_API_URL` dokümantasyonu ve örnek değer kontrol edilmeli (geliştirici kurulumu).

### Estetik ve UX
- [ ] **Inline `<style>` blokları:** Birçok sayfada büyük `dangerouslySetInnerHTML` CSS — modüler CSS/Tailwind veya paylaşılan modüllere taşınmalı (bakım zorluğu).

### Teknik borç

- [ ] **Bağımlılıklar:** `recharts`, `framer-motion` kısmen kullanılıyor / ölü import riski — bundle gözden geçirilmeli.
